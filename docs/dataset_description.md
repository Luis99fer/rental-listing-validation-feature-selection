# Dataset Description

## Overview

This project uses the dataset from the Kaggle competition **Two Sigma Connect: Rental Listing Inquiries**.

The dataset contains rental apartment listings and is used to predict the level of interest that each listing receives.

The target variable is:

```text
interest_level
```

Typical target classes are:

```text
low
medium
high
```

## Source

The dataset comes from the original Kaggle competition:

```text
https://www.kaggle.com/competitions/two-sigma-connect-rental-listing-inquiries
```

## Problem Type

This is a supervised machine learning problem focused on rental listing interest prediction.

```text
Input: Rental listing attributes
Output: Interest level
Target variable: interest_level
```

The model uses listing characteristics, location, price, photos, description, and amenity features to estimate user interest.
