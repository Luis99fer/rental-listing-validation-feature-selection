# Dataset Access

## Overview

This project uses the dataset from the Kaggle competition **Two Sigma Connect: Rental Listing Inquiries**.

The task is to predict the level of interest generated by rental apartment listings based on listing attributes, location, price, photos, description, and extracted amenity features.

The target variable is:

```text
interest_level
```

Typical classes are:

```text
low
medium
high
```

## Source

The dataset is available from the original Kaggle competition:

```text
https://www.kaggle.com/competitions/two-sigma-connect-rental-listing-inquiries
```

## Main Files

The original Kaggle dataset includes files such as:

```text
train.json
test.json
sample_submission.csv
```

## Target Variable

The target variable is:

```text
interest_level
```

It represents the level of user interest in a rental listing.

The problem can be treated as a supervised machine learning task where the model learns from historical listings and predicts interest level for unseen listings.

## Dataset Fields

The dataset contains rental listing information such as:

```text
listing_id

```

These features are used to transform unstructured listing amenities into machine learning-ready numerical indicators.
